# Scifi Dungeon Crawler Server
Tile based dungeon crawler multiplayer game server and [client](https://github.com/leolastikka/scifi-dungeon-crawler-client).

Requirements:
* node version 18.16.0 lts (possibly works with older)
* npm

Development:
```
npm install
npm run dev
```

Production:
```
npm install
npm run build
npm run start
```
